## Use

`kure stats`

## Description

Show database statistics.

## Flags 

|  Name     |     Type      |    Default    |              Description                |
|-----------|---------------|---------------|-----------------------------------------|
| json      | bool          | false         | Output statistics in JSON format        |

### Examples

Show statistics:
```
kure stats
```

Show statistics in JSON format:
```
kure stats --json
```
